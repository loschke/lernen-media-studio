# Bildwerkstatt

## Generieren
- Prompt Input mehr höhe zum Eingeben, Scrollbar bei sehr langen Prompts bitte dezenter gestalten
- Ein zusätzliche Iconbutton für Settings neben dem Prompt Input ergänzen, der die Settings öffnet
- Zu den Settings gehören: Seitenverhältnis und die Modellauswahl
- Seitenverhältnis standardmäßig auf 1:1 setzen
- Folgende Modelle sollen zur Auswahl stehen:
    - gemini-2.5-flash-image -> Titel: Standardmodell -> Default Setting
    - gemini-3.1-flash-image-preview -> Titel: Schnell & Kreativ
    - gemini-3-pro-image-preview -> Titel: Höchste Qualität
- Die Anzeige des generierten Bildes bitte immer so groß wie möglich im Viewport anzeigen, den Viewport ausnutzen und bestmöglich ausfüllen ohne eine Scrollbar zu erzeugen- fit in viewport sozusagen

## Bearbeiten
- Startscreen ist so ok
- Die Auswahl eines Bildes aus der Galerie oder per Upload führt aktuell dazu, dass es Mittig im Viewport angezeigt wird und sich das Prompt Input sichtbar wird. Das ist nicht optimal
- Aktuell kann ich auch nur ein Bild auswählen, folgende Möglichkeiten zur Auswahl von Startbildern sollen zur Verfügung stehen:
    - Maximal 3 Bilder können als Start Referenzen ausgewählt werden
    - Die Auswahl kann teilweise oder vollständig per Upload erfolgen
    - Die Auwswahl kann teilweise oder vollständig aus der Galerie erfolgen
    - Kombination aus Upload und Galerie ist möglich
- Die ausgewählten Bilder sollen im Umfeld des Prompt Inputs angezegit werden und einfache per X Button wieder entfernt werden können
- Das Prompt Inputfeld kann auch zum Startansicht des Bearbeiten-Modus angezeigt werden.
